# 파일 양식

## main 폴더의 파일 양식
- 예시 : `2201_1229_population.csv`
- `2201`
  - 맨 앞 숫자 `2` (용도)
    - 1 : 프로그램
    - 2 : 데이터
    - 3 : 문서
  - 두번째 숫자 `2` (작성자, 가나다순)
    - 1 : 김선중
    - 2 : 김연수
    - 3 : 배현우
    - 4 : 이승준
    - 5 : 차윤겸
  - 세번째, 네번째 숫자 `01` : 임의번호
- `1229` : 작성날짜
- `population.csv` : 파일명 (description)
